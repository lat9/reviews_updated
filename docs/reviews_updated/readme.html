What it does

This plugin provides updates to the standard Zen Cart processing of reviews for your products, enabling reviews to be written by guests — controlled by an admin configuration value.

These standard zen-cart database configuration values are used by this plugin:

  • Configuration->Minimum Values->Product Review Text
  • Configuration->Product Info->Product Reviews Require Approval
  • Configuration->E-mail Options->Send Notice of Pending Reviews Emails To - Status
  • Configuration->E-mail Options->Send Notice of Pending Reviews Emails To

These database configuration values are added by this plugin:

  • Configuration :: Minimum Values :: Product Review Write - Guest Reviewer Name (default: 5)
  • Configuration :: Product Info :: Enable product reviews by guests? (default: 0/false)

Installation

This plugin requires storefront template overwrites, but modifies no Zen Cart core-files; you should always backup your cart’s database and files prior to making any changes.

Using the plugin's unzipped zip-file contents as a basis:

  1. Rename the YOUR_ADMIN directory to match your site's secret admin directory name.
  2. Rename the /includes/languages/english/YOUR_TEMPLATE directory, changing YOUR_TEMPLATE to match your active template's name.
  3. If your site uses a template based on the in-built responsive_classic template:
    1. Rename the /includes/templates/YOUR_RC_TEMPLATE directory, changing YOUR_RC_TEMPLATE to match you site's template's name.
    2. Delete the /includes/templates/YOUR_BOOTSTRAP_TEMPLATE directory, since the file in that directory isn't used.
    3. If you have made changes to the template's tpl_product_reviews_write_default.php module, you'll need to merge the one (1) marked change section in the plugin's version with your previous changes.
  4. If your site uses a template based on the bootstrap template:
    1. Rename the /includes/templates/YOUR_BOOTSTRAP_TEMPLATE directory, changing YOUR_BOOTSTRAP_TEMPLATE to match you site's template's name.
    2. Delete the /includes/templates/YOUR_RC_TEMPLATE directory, since the file in that directory isn't used.
    3. If you have made changes to the template's tpl_product_reviews_write_default.php module, you'll need to merge the one (1) marked change section in the plugin's version with your previous changes.
  5. Copy the files in the plugin's /includes and renamed /YOUR_ADMIN directory to your site's file-system.
  6. Login to (or refresh the screen of) your Zen Cart admin. You'll see the two configuration settings added by the plugin's admin auto-installer.
  7. Review Configuration :: Minimum Values :: Product Review Write - Guest Reviewer Name, changing that minimum length if needed.
  8. Go to Configuration :: Product Info :: Enable product reviews by guests? to enable reviews by guests.

Un-install

To un-install the plugin:

  1. Delete the two files in your admin directory:
    1. /includes/auto_loaders/config.reviews_updated_admin.php
    2. /includes/init_includes/init_reviews_updated_admin.php
  2. Copy the contents of /docs/reviews_updated/uninstall/reviews_updated_uninstall.sql, paste that SQL statement into your admin's Tools :: Install SQL Patches and run the SQL command.
  3. The storefront additions for the plugin won't interfere with your storefront operations without the configuration settings just removed, but can be removed for completness:
    1. /includes/classes/observers/auto.reviews_updated.php
    2. /includes/languages/english/YOUR_TEMPLATE/lang.product_reviews_write.php
    3. /includes/templates/YOUR_TEMPLATE/templates/tpl_product_reviews_write_default.php, backing out the changes made by the plugin.
